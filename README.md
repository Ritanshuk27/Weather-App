# 🌤️ Weather App

A modern and responsive **Weather Application** built using **HTML, CSS, and JavaScript**. This app provides real-time weather updates based on city search or the user’s current location.

---

## 🚀 Features

* 🔎 Search weather by city name
* 📍 Auto-detect location using Geolocation API
* 🌡️ Real-time temperature, humidity, and weather conditions
* 🌙 Dark mode toggle
* 🕸️ Animated background using particles.js
* ⚡ Fully responsive designs

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & responsiveness
* **JavaScript (Vanilla)** – Logic & API handling
* **OpenWeatherMap API** – Weather data
* **Geolocation API** – Location detection
* **particles.js** – Background animation

## ⚙️ Installation & Setup

1. Clone the repository

   ```bash
   git clone https://github.com/Ritanshuk27/Weather-App.git
   ```

2. Navigate to the project folder

   ```bash
   cd Weather-App
   ```

3. Add your API key in `script.js`

   ```javascript
   const API_KEY = "your_api_key_here";
   ```

4. Open `index.html` in your browser
